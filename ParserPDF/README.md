## Clear PDF Parser
#### Recebe de input 1 ou vários arquivos PDF "Nota de Negociação" da Clear e extraí os dados (venda, compra, ação, dia, valor total) de cada nota.
#### Requisitos:
```pip3 install pdfminer```
